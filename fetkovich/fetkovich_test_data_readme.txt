
Fetkovich方法测试数据说明
========================

本Excel文件包含用于测试Fetkovich产量递减曲线分析工具的模拟数据。

文件结构:
---------
- 每个工作表对应一种特定的递减类型
- "所有数据汇总"工作表包含所有情况的产量数据
- 每个工作表包含三列:
  * 时间（天）: 从0到1000天的生产时间序列
  * 产量（m³/d）: 添加了随机噪声的实际产量数据
  * 理论产量（m³/d）: 无噪声的理论产量数据（用于参考）

包含的递减类型:
---------------
1. 指数递减类型 (b < 0.3):
   - 指数递减_高压气井: b=0.1, 初期递减快
   - 指数递减_常规气井: b=0.2

2. 双曲递减类型 (0.3 ≤ b < 0.9):
   - 双曲递减_类型1: b=0.3
   - 双曲递减_类型2: b=0.5
   - 双曲递减_类型3: b=0.7

3. 调和递减类型 (b ≈ 1.0):
   - 调和递减_类型1: b=1.0
   - 调和递减_类型2: b=1.0

4. 实际模拟案例:
   - 实际案例_气井A: b=0.4
   - 实际案例_气井B: b=0.6

使用说明:
---------
1. 在Fetkovich分析工具中上传此Excel文件
2. 选择任意一个工作表
3. 选择"时间（天）"作为时间列
4. 选择"产量（m³/d）"作为产量列进行拟合分析
5. 比较不同递减类型的拟合效果和参数估计准确性

技术参数:
---------
- 数据点数: 100个
- 时间范围: 0-1000天
- 噪声水平: 3-8%的随机波动
- 初始产量(q_i): 400-1500 m³/d
- 初始递减率(d_i): 0.02-0.15
- 递减指数(b): 0.1-1.0

注意事项:
---------
- 数据已添加随机噪声以模拟实际生产数据的波动
- "理论产量"列可用于验证拟合结果的准确性
- 建议先使用单个工作表进行测试，熟悉后再尝试其他类型
